# From Model to Production — Fraud Detection (PyData Lagos)

This repository contains materials for a **2-part hands-on session** at PyData Lagos:

> **From Business Problem → Machine Learning Model → Production API**

---

##  Overview

In this series, we walk through the full journey of a machine learning solution:

* Understand a real-world fraud detection problem
* Build and evaluate a machine learning model
* Deploy the model as an API using FastAPI
* Interact with the model like a real system

 The focus is on **making models usable**, not just building them.

---

##  Sessions

### 🟦 Session 1 — ML Model

* Problem framing
* Imbalanced data
* Logistic Regression
* Precision, Recall, F1-score
* Class weights

---

### 🟩 Session 2 — Deployment

* APIs and FastAPI
* Serving a model
* Running an API
* Testing via `/docs` and Python

---

## 🔄 The Big Picture

**Problem → Data → Model → Production**

 This is what real-world machine learning looks like.
